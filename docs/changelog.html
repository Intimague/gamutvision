Gamutvision™ change log (release notes)

You can download an evaluation version of Gamutvision that allows up to 20 ICC profiles to be analyzed. Online purchase and registration is available. The purchase price includes one year of upgrades.

Date Release Description
Jan. 26, 2007 1.3.3 Fixed a bug that prevented Gamutvision from starting when a profile used in a previous run was deleted or had its name changed.
Jan. 16, 2007 1.3.2 Fixed a bug that terminated Print Test prematurely.
Jan. 2, 2007 1.3.1 Optionally displays the gamut volume on the 3D L*a*b* plot. This is the single number that best characterizes color response.
Jan. 1, 2007 1.3 Changes in image colors due to gamut mapping or printing can now be plotted in 3D L*a*b* space. Both color points and vectors can be displayed, and they can be superimposed on wireframe displays of the input or output color gamut. A simulated GretagMacbeth™ ColorChecker® can be analyzed in a similar manner to input images. The Reset view button no longer affects screen size or position. The screen can be reset by means of the Reset screen command in the Settings dropdown menu.
Dec. 27, 2006 1.2.4 Profiles can now be selected from a list of recently-opened profiles (in addition to browsing or selecting the last profile).
Nov. 25, 2006 1.2.3 Fixed some bugs that could crash Gamutvision with CMYK profiles. Fixed an issue with displays when color differences were zero.
Nov. 18, 2006 1.2.2 Fixed the Delta-Hue angle (Δ Hue (°)) display in Image color difference to eliminate large errors for dark or low chroma colors. Added Δ Hue (°) to the probe display.
Oct. 16, 2006 1.2.1 Fixed a bug that entered with 1.1.9 or 1.2 that prevented Image Color Difference from running.
Oct. 15, 2006 1.2 Black Point Compensation can be specified separately for the Left (1>2) and Right (3:4) data flows. Fixed a Purchase button bug that linked to the Imatest Light purchase page.
Oct. 10, 2006 1.1.9 The Black & White density response, Output vs. Input L* a* b* c* display now includes the L* response for primary colors RYGCBM with saturation S(HSL) set by the saturation slider used in several other displays. Displays now update more quickly.
Oct. 8, 2006 1.1.8 Adds soft-proofing rendering intents, allowing comparision of soft-proofed (monitor) gamuts with the actual gamuts. Fixes some issues in updating plots when profile 3 is set to the output of box 2.
Oct. 5, 2006 1.1.7 3D views (zoom and angle) can now be saved and retrieved in named (.ini) files using Save and Retrieve 3D View in the Settings menu. Occurrences of a Matlab bug that can cause the legend to disappear in some 2D plots have been minimized.
Oct. 4, 2006 1.1.6 The zoom setting is retained for similar displays. Clicking any of the View buttons does not reset the zoom. This facilitates comparison of profile details. A "Color operator" warning message in the DOS window has been eliminated.
Sept. 10, 2006 1.1.5 An option has been added to the 3D L*a*b* More dialog box to turn off OpenGL rendering in the 3D L*a*b* display. This is mostly useful for diagnosing mysterious crashes. OpenGL can cause instabilities, but it's needed for the transparent wireframe display.
Sept. 8, 2006 1.1.4 Saturation S (default = 1) can be adjusted in the 3D L*a*b* display.
Aug. 28, 2006 1.1.3 The test pattern image (upper right) for the 2D a*b* displays has been corrected. A new page on smudging caused by a profile problem has been added.
Aug. 27, 2006 1.1.2 Several small enhancements to the 3D/2D color difference display that make it easier to diganose problems inside the gamut boundary (S(HSL) < 1). h* axis labels have been corrected.
Aug. 22, 2006 1.1.1 The B&W density response includes an output L*, a*, b*, c* vs input L* plot that shows the grayscale pattern chroma (c*): a strong indicator of B&W print neutrality.
Aug. 21, 2006 1.1 HSL Lightness (L) can be varied from its default value of 0.5 in the 2D a*b* saturation and color difference plots. All interior regions of the ICC profile may now be analyzed.
Aug. 15, 2006 1.0.3 The HSL saturation (S) of the test pattern can be varied in 3D/2D color difference displays only. 3D/2D color difference displays have been improved in several aspects, with some bugs have been fixed. Displays with H*L* backgrounds (derived from L*a*b*) are much improved. Print Test output has been simplified with the removal of a superfluous figure.
Aug. 7, 2006 1.0.2 A Probe for displaying several color difference metrics has been added to the Image color difference display. 3D HL and Image color difference displays are now scaled to the maximum value of current and recent displays. This facilitates performance comparisions for different profiles. Scaling can be reset for the current image by pressing Reset scale, selecting a new difference metric, or opening a new image file (for Image color difference).
Aug. 4, 2006 1.0.1 Profile window highlighting is updated correctly when Display A and B are updated. A missing file issue with registration has been resolved.
July 23, 2006 1.0 Gamutvision is quietly released. There will be more fanfare after the website is enhanced and a press release is prepared.
 
June 1, 2006 Beta 1.0.14 "Workflow 1 and 2" has been changed to "Display A and B" to more accurately represent its function. Several minor bugs have been fixed.
May 14, 2006 Beta 1.0.13 Black point compensation now works (Colorimetric, Saturation intents only).
May 7, 2006 Beta 1.0.12 Fixed some problems with Save figure. Fixed the 3D lighting again (maybe got it right, finally).
April 30, 2006 Beta 1.0.11 The evaluation version has been implemented, but since Gamutvision does not have its own registration database, it uses Imatest's.
April 24, 2006 Beta 1.0.10 Adds the capability of analyzing images for perceptual differences that arise from gamut mapping: you can see how much an image will change when sent to a printer.
April 22, 2006 Beta 1.0.9 Color differences can now be plotted in 3D. Lighting now appears on 3D plots the first time they are viewed.
April 17, 2006 Beta 1.0.8 Options have been added to the 3D wire frame view, using the More button. Two new plots have been added, showing color differences (ΔE*ab, ΔC*ab, ΔE*94, ΔC*94, ΔE*CMC, and ΔC*CMC ) on the a*b* and L-H planes.
April 2, 2006 Beta 1.0.7 Round trip has been corrected: it now conforms to the usual definition: Lab > profile > Lab. The chart image (upper right) has been corrected to show correct black levels when viewing output mapped to the monitor. There have been many small interface improvements.
March 29, 2006 Beta 1.0.6 B&W density plots now have three options. Vectors have been added to the 2D a*b* plots.
March 26, 2006 Beta 1.0.5 2D HSL plots (similar to Imatest Print Test) have been added.
March 24, 2006 Beta 1.0.4 A 2D u'v' plot (similar to the 2D xy plot) has been added.
